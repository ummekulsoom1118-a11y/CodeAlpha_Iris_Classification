# Iris Flower Classification 🌸

This project uses Machine Learning to classify iris flowers into three species:
- Setosa
- Versicolor
- Virginica

## 📊 Technologies Used
- Python
- Pandas
- Scikit-learn

## ⚙️ Model Used
- Decision Tree Classifier

## 📈 Result
The model achieved high accuracy in predicting flower species.

## 📌 Conclusion
This project demonstrates basic classification using machine learning.
